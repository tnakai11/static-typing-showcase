# Static Typing Showcase

A modern, animated web page showcasing the benefits of statically typed programming languages.

## Features
- **Modern Design**: Dark mode, glassmorphism, and neon accents.
- **Rich Animations**: Powered by GSAP and ScrollTrigger.
- **Interactive Elements**: Hover effects and code comparisons.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (ES6+)
- [GSAP](https://greensock.com/gsap/) (Animation Library)
